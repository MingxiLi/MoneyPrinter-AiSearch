# 👋 简介
一个AI驱动的自动短视频生成器
只需提供一个视频 <b>主题</b> 或 <b>关键词</b> ，自动生成文案，根据文案自动配置素材、字幕、音乐，最终合成短视频。

# 🗞️ 更新内容
- 2024.04.16: 更新了<b>AI素材搜索<b>功能
    - 提升了素材和文案的匹配度（Pexels自带搜索功能不准确）
    - 更新了中文AI搜索模型和相关数据库，支持Pexels上面近40w视频检索
